Panduan Aplikasi Bell Sekolah Otomatis

Aplikasi ini dirancang khusus untuk mempermudah tugas staf Tata Usaha (TU) Sekolah Dasar. Aplikasi berjalan 100% di dalam browser (Chrome, Firefox, Edge) tanpa perlu koneksi internet dan tanpa perlu instalasi database rumit.

🚀 Cara Instalasi / Penggunaan Pertama

  1. Unduh (Download) file HTML utama dari tampilan di atas.
  2. Simpan file tersebut di komputer atau laptop sekolah Anda (misalnya di Desktop) dengan nama BellSekolah.html.
  3. Cukup Klik Ganda (Double-click) file tersebut. Aplikasi akan otomatis terbuka di browser default Anda.
  4. Jadikan file ini Bookmark agar lebih mudah diakses keesokan harinya.

✨ Fitur-Fitur Utama

  • Tanpa Internet: Semua data jadwal dan file suara disimpan dengan aman di memori browser (LocalStorage & IndexedDB) komputer Anda.
  • Sintesis Suara Pintar: Walaupun Anda tidak memiliki/mengupload file MP3, aplikasi ini bisa menghasilkan nada bell sendiri yang jernih (Bell Masuk 3 Nada, Istirahat 2 Nada, Pulang Westminster).
  • Keamanan Refresh: Jika tanpa sengaja tertutup, data Anda tetap aman karena telah tersimpan di browser.

📝 Panduan Operasional

1. Memulai Sistem Bell (Setiap Pagi)

  • Buka aplikasi.
  • Di menu Dashboard Utama, klik tombol biru besar "Mulai Sistem Bell".
  • Indikator akan berubah menjadi hijau ("Sistem Aktif & Memantau"). Biarkan tab browser ini tetap terbuka selama jam sekolah berlangsung.
  • Opsional: Anda bisa menekan tombol "Fullscreen" agar tampilan jam membesar memenuhi layar.

2. Mengatur Jadwal

  • Buka menu Pengaturan Jadwal.
  • Pilih hari (misal: Senin).
  • Klik tombol biru "+ Tambah Jadwal".
  • Masukkan jam (contoh: 07:15), nama kegiatan, jenis bell, lalu simpan.
  • Aplikasi akan otomatis berbunyi sesuai hari dan jam di komputer Anda.

3. Mengganti Suara Bell dengan MP3 Sendiri

  • Jika sekolah Anda memiliki file lagu atau suara rekaman sendiri (MP3/WAV), buka menu Library Suara.
  • Pilih file di bagian "Upload Suara Kustom" dan klik Simpan Suara.
  • Sekarang, saat Anda menambah/mengedit jadwal di menu Pengaturan Jadwal, suara MP3 Anda akan muncul di bagian pilihan "Sumber Suara".

4. Tombol Darurat (Manual)

  • Jika ada perubahan jadwal mendadak (misalnya istirahat dipercepat), Anda tidak perlu mengubah jadwal.
  • Buka menu Manual Panel dan klik salah satu tombol bell raksasa yang tersedia. Suara akan langsung berbunyi.

5. Memindahkan Data ke Komputer Lain

  • Buka menu Pengaturan Sekolah.
  • Klik "Export Data Jadwal". Akan terdownload sebuah file .json.
  • Di komputer baru, buka aplikasi yang sama, klik "Import" dan pilih file .json tersebut. Jadwal Anda akan otomatis kembali seperti semula!
