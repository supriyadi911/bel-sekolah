Panduan Aplikasi Bell Sekolah Otomatis

Aplikasi ini dirancang khusus untuk mempermudah tugas staf Tata Usaha (TU) Sekolah Dasar. Aplikasi berjalan 100% di dalam browser tanpa perlu koneksi internet.

🚀 Cara Instalasi / Penggunaan Pertama

  1. Unduh (Download) file HTML utama dari tampilan di atas.
  2. Simpan file di komputer sekolah Anda (misal di Desktop) dengan nama BellSekolah.html.
  3. Klik Ganda (Double-click) file tersebut untuk membukanya di browser.
  4. Jadikan file ini Bookmark agar lebih mudah diakses.

✨ Fitur-Fitur Utama

  • Tanpa Internet: Semua data disimpan dengan aman di memori browser (LocalStorage).
  • Sintesis Suara: Aplikasi ini bisa menghasilkan nada bell sendiri yang jernih otomatis.
  • Responsif Layar HP: Dapat dibuka di tablet atau HP sebagai panel kontrol tambahan (jika file dicopy ke HP).

📝 Panduan Operasional

1. Memulai Sistem Bell

  • Buka aplikasi, di menu Dashboard Utama, klik "Mulai Sistem Bell".
  • Biarkan tab browser ini tetap terbuka selama jam sekolah.

2. Mengatur Jadwal (Import Excel)

  • Buka menu Jadwal.
  • Klik 📥 Template Excel, edit filenya lalu simpan.
  • Klik 📂 Import Excel dan pilih file yang sudah diedit. Jadwal seminggu langsung terpasang otomatis.
